# Conductor Dashboard

Unified web interface for the prductr orchestration platform.

## Overview

Next.js dashboard for monitoring and managing multi-tenant orchestration:
- Agent management
- Task monitoring
- Cost analytics
- Access control

## Deployment

- **Production**: https://conductor.prductr.com
- **Platform**: Vercel + Cloudflare D1

## Development

```bash
npm run dev
```

Runs on port 3100.

## Features

- Real-time agent status
- Task queue visualization
- Cost tracking & budgets
- Sandbox management (E2B)
- Secret management (encrypted)
