# @prductr/lisa

Archaeological rescue & migration agent for abandoned projects.

## Overview

Lisa reconstructs lost context through git archaeology, timeline analysis, and mission extraction to revive dead codebases.

## Features

- Git timeline reconstruction
- Semantic memory extraction
- Bead & convoy work structuring
- Multi-project reconciliation

## Usage

```bash
/lisa:rescue    # Full rescue pipeline
/lisa:discover  # Extract semantic memory
/lisa:plan      # Generate roadmap
/lisa:structure # Create work items
```
