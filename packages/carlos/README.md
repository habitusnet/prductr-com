# @prductr/carlos

AI-powered product roadmap generation agent.

## Overview

Carlos scans docs, PRDs, and legacy tasks to create comprehensive scopecraft roadmaps with epics, stories, and PMF metrics.

## Features

- Auto-generate 6 scopecraft files
- Vision & stage definition
- Risk & dependency mapping
- PMF validation framework

## Usage

```bash
/carlos:roadmap       # Generate full roadmap
/carlos:market-fit    # Assess product-market fit
```

## Output

Creates `./scopecraft/` with 6 files:
1. VISION_AND_STAGE_DEFINITION.md
2. ROADMAP.md
3. EPICS_AND_STORIES.md
4. RISKS_AND_DEPENDENCIES.md
5. METRICS_AND_PMF.md
6. OPEN_QUESTIONS.md
